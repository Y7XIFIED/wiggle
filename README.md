# Wiggle

> A playful interactive landing page with smooth motion and character-driven UI.

![License](https://img.shields.io/badge/license-MIT-black.svg)
![Version](https://img.shields.io/badge/version-1.0.0-5546ff.svg)
![Build](https://img.shields.io/badge/build-static%20site-success.svg)

## Features

- Responsive single-page hero experience
- Mouse-tracking emoji sphere with GSAP easing
- Shuffle typography interactions on hover/focus
- Custom type + bold visual direction
- Clean static structure, easy to host anywhere

## Installation

```bash
git clone https://github.com/Y7XIFIED/REPO_NAME.git
cd REPO_NAME
```

No dependency install needed. This project runs as a static website.

## Usage

Open `index.html` directly in a browser, or serve locally:

```bash
# Python
python -m http.server 8080

# Node (optional)
npx serve .
```

Then visit `http://localhost:8080`.
